
### **🔄 Code Translation Demo:**
```python
# Input: Python
def fibonacci(n):
    if n <= 1:
        return n
    return fibonacci(n-1) + fibonacci(n-2)
```

**→ Translates to JavaScript in 3 seconds →**

```javascript
function fibonacci(n) {
  if (n <= 1) {
    return n;
  }
  return fibonacci(n-1) + fibonacci(n-2);
}
```
